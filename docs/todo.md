# Prism Vision 

* Using NextJS and Typescript for the Frontend, 
And FastAPI for the backend, 

Build a multimodal computer vision and VLM full stack web application that performs instance segmentation on 
The predicted objects. 

Working Principle: 
Develop a docx file explaining the whole process of this project. 

The Objectives will be 
* Users can log in 
* Users will have a landing page explain the work flow of the project 
* Users can register 
* Users will have a dashboard page that will be responsible for performing the ML operations. 
* Users will have a history of all the ML operations performed. 
* We will use MongodDB database for storing the predicted results 
* Users will perform multimodal computer vision and VLM inference on an uploaded image, the result will show both the inference of the VLM model and the display of the instance segmented region. 
* We will use Ultralytics instance segmentation model for this. (Yolo).
